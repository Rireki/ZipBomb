# ZipBomb
Turning my temperature table program into a zipbomb generator.
# How to
1. Place zipbomb.exe to your computer. Example, D:
2. Open CMD.
3. Navigate to D:
4. Run this command;
   zipbomb >zipbomb.txt
5. Make sure to turn it off before it fills your hard drive!
6. Use your compression software of your choice to compress the resulting text file.
